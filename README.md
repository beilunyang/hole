# hole
另一个使用Vue开发的Ghost博客主题    

### 使用方法
* 将hole放入ghost themes文件夹中
* 开启ghostAPI
* 在后台设置每页显示博文数为5
* 如需在首页显示文章摘要，请确保填写搜索引擎优化中的优化摘要一栏；如需显示博文封面，请填写为博文设置图片一栏
